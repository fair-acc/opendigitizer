# Flowgraph

Visual styling mockups for flowgraph are available in ../theme-design

## Features

- filter source/sink to focus flowgraph
- filter/list/add blocks
- filter/list to add sources (also availble in Chart UI)
- move and connect elements/blocks
- name/add sinks
- view/modify FEC of a source / modify block parameters (also availble in Chart UI)
- delete blocks
- filter/add hierarchical blocks
- define/save hierarchical blocks
- auto-layout blocks
- break up hierarchical blocks.
